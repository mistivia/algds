# Type-safe Generic Algorithms and Data Structures for C

- Generic red-black tree
- Generic vector
- Generic linked-list
- Generic array-based linked list (AList)
- Generic priority queue
- Generic Hash Table
- Generic quick sort
- Augmented red-black tree
- Murmur Hash
- String utilities
- Smart pointers (Box, Arc)

## Build & Install

    make
    make install

## Run Tests

    make test
